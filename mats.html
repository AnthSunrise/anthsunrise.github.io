Planificador de Armadura Pesada - Lineage 2 (Hasta Dynasty)

Progresión de armaduras:
D: Tier 1 → Tier 2
C: Tier 1 → Tier 2 → Tier 3
B: Zubei → Avadon → Blue Wolf → Doom
A: Dark Crystal → Tallum → Nightmare → Majestic
S: Grado S (Armor/Helmet/Gloves/Boots)
Dynasty: Armadura final

Instrucciones: Selecciona el objetivo que quieres alcanzar y la armadura que ya tienes equipada. El sistema calculará automáticamente los materiales que necesitas para avanzar desde tu armadura actual hasta tu objetivo.

Imágenes: Para añadir imágenes a los materiales, simplemente añade archivos con el nombre del material en la carpeta "icons" (ej: "Adena.png", "Crystal D.png"). Las imágenes deben ser de 32x32 píxeles.

Materiales Requeridos

Nota sobre imágenes: Las imágenes se cargan desde la carpeta "icons". Si no hay imagen para un material, se mostrará un icono por defecto. Para añadir tus propias imágenes, crea una carpeta llamada "icons" en el mismo directorio que este archivo HTML y añade imágenes PNG de 32x32 con el nombre exacto del material.
